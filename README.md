# Beleg Verteile Systeme


## Code kompilieren
```
cd src
javac bootstrap/*.java  mandelbrot/*.java
```
<br>

## Bootstrap starten:
```
java bootstrap/BootstrapRegistration
```
## Server starten:

```
java mandelbrot/MandelbrotServer <ip oder localhost> + <servername>
```
